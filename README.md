# 衣類管理システム (Wardrobe Manager)

React ベースの個人用衣類管理アプリで、ユーザーが自分の衣類を効率的に管理できるようサポートします。

## 機能の特徴
- 衣類画像のアップロードとプレビュー  
- 衣類の分類管理とフィルタリング機能  
- タグ付けシステム  
- コーディネートの組み合わせ提案機能  
- レスポンシブデザイン  

## 技術スタック
- React  
- React Router  
- Tailwind CSS  
- Lucide Icons
