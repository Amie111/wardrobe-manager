# 衣橱管理系统 (Wardrobe Manager)

一个基于 React 的个人衣橱管理应用，帮助用户更好地管理自己的衣物。

## 功能特点

- 衣物图片上传与预览
- 衣物分类管理
- 标签系统
- 筛选功能
- 响应式设计

## 技术栈

- React
- React Router
- Tailwind CSS
- Lucide Icons

## 开始使用

1. 克隆项目
   git clone https://github.com/你的用户名/wardrobe-manager.git
2. 安装依赖
   npm install
3. 启动开发服务器
   npm start
